Parameters illustration of Lineage client interface in conf/interface.au3
file.

1. kTargetWindowLeft - this is left-up point of the target window
2. kTargetWindowRight - this is right-bottom point of the target window
3. kTargetWindowColorBrown - this is color of random pixel in the target window
4. kTargetWindowColorGray - this is color of random pixel in the target window
5. kTargetHealthLeft - this is left-up point of the target's HP bar
6. kTargetHealthRight - this is right-bottom point of the target's HP bar
7. kTargetHealthColor - this is color of the target's full HP bar (by default red)
8. kTargetManaLeft - this is left-up point of the target's MP bar
9. kTargetManaRight - this is right-bottom point of the target's MP bar
10. kTargetManaColor - this is color of the target's full MP bar (by default blue)
11. kTargetManaEmptyColor - this is color of the target's empty MP bar (by default dark blue)
12. kSelfHealthLeft - this is left-up point of the player's HP bar
13. kSelfHealthRight - this is right-bottom point of the player's HP bar
14. kSelfHealthColor - this is color of the player's full HP bar (by default red)
15. kSelfManaLeft - this is left-up point of the player's MP bar
16. kSelfManaRight - this is right-bottom point of the player's MP bar
17. kSelfManaColor - color of the player's full MP bar (by default blue)
