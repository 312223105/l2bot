L2Bot 0.6 version
=================

ВВЕДЕНИЕ
--------

L2Bot - это набор AutoIt скриптов, реализующих ботов разного назначения для Lineage 2.

УСТАНОВКА
---------

Для запуска L2Bot вам необходимо скачать интерпретатор языка AutoIt с
официального сайта:
http://www.autoitscript.com/site/autoit/downloads

Если вы используете 64-разрядную версию Windows, выберете 64-разрядную версию
интерпретатора в диалоге установки.

Скачайте архив со скриптами L2Bot и распакуйте его.

НАСТРОЙКА
---------

Прежде всего поставьте галочку "чат с Enter" в настройках вашего клиента
Lineage.

Вы можете модифицировать перечисленные ниже конфигурационные файлы, чтобы лучше
адаптировать L2Bot для вашего клиента Lineage.

Список конфигурационных файлов из каталога 'conf':
hotbar.au3 - содержит настройку панели горячих клавишь
interface.au3 - расположение окон и областей экрана для анализа цветов
control.au3 - специальные клавищи управления персонажем
targets_ru.au3 - список шаблонов для поиска мобов по имени

Следующая информация может быть указана в файле hotbar.au3:
1. kAttackKey - клавиша действия /attack на панели горячих клавишь (по умолчанию F1)
2. kNextTargetKey - клавиша действия /targetnext на панели (по умолчанию F10)
3. kPickDropKey - клавиша действия /pickup на панели (по умолчанию F8)
3. kSitKey - клавиша сесть/встать на панели (по умолчанию F9)
6. kHealthPoition - клавиша использования зелья исцеления (по умолчанию F5)

Для уточнения следующей информации об интерфейсе воспользуйтесь программой CoolPix.exe из
каталога 'tools'.

Следующая информация может быть указана в файле interface.au3:
1. kTargetWindowPos[x1, y1, x2, y2] - координаты левой верхней точки (x1 и y1) и
правой нижней точки (x2 и y2) окна состояния цели
2. kTargetWindowColorBrown - первый цвет для определения наличия окна цели
(по умолчанию коричневый)
3. kTargetWindowColorGray - второй цвет для определения наличия окна цели (по
умолчанию серый)
4. kTargetHealthColor - цвет полной полосы HP цели
5. kTargetManaColor - цвет полной полосы MP цели (для петов/самонов)
6. kTargetManaEmptyColor - цвет пустой полосы MP цели (для петов/самонов)
7. kTargetHealthPos - область полосы HP цели для обнаружения события "цель
повреждена"
8. kSelfHealthCritPos - область полосы HP игрока для обнаружения события
"критическое значение HP"
9. kSelfHealthHalfPos - область полосы HP игрока для обнаружения события
"осталось половина HP"
10. kSelfHealthEmptyPos - область полной полосы HP игрока
11. kSelfHealthColor - цвет полной полосы HP игрока
12. kSelfManaMinPos - область полосы MP игрока для обнаружения события
"критическое значение MP"
13. kSelfManaColor - цвет полной полосы MP игрока
14. kMoveControlPos1 - первая точка на экране для обнаружения события "игрок
двигается" (это произвольная точка на экране не занятая каким-либо окном)
15. kMoveControlPos2 - вторая точка на экране для обнаружения события "игрок
двигается" (это произвольная точка на экране не занятая каким-либо окном)
16. kMoveControlPos3 - третья точка на экране для обнаружения события "игрок
двигается" (это произвольная точка на экране не занятая каким-либо окном)

Также вы можете указать горячие клавиши специфичных для каждого класса умений.
Эти настройки вы можете найти в следующих файлах каталога 'tactics' -
spoiler.au3, tank.au3 и warcrayer.au3.

ЗАПУСК
------

Для запуска L2Bot выполните следующие действия:

1. Запустите ваш клиент Lineage в оконном режиме (нажмите Alt+Enter для перехода
в этот режим из полноэкранного) и зайдите в игру своим персонажем.
2. Перейдите в локацию с мобами
3. Сверните окно вашего клиента Lineage
4. Запустите один из скриптов spoiler.au3, warcrayer.au3 или tank.au3 из
корневого каталога проекта L2Bot
5. Переключитесь и разверните окно клиента Lineage
6. Нажмите Alt+F2 чтобы бот захватил окно клиента Lineage
7. Нажмите Alt+F3 для запуска бота
8. Нажмите Alt+F1 для остановки бота
