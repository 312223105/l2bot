L2Bot версия 0.8
=================

ВВЕДЕНИЕ
--------

L2Bot - это набор AutoIt скриптов, реализующих ботов разного назначения для Lineage 2.

УСТАНОВКА
---------

Для запуска L2Bot вам необходимо скачать интерпретатор языка AutoIt с
официального сайта:
http://www.autoitscript.com/site/autoit/downloads

Если вы используете 64-разрядную версию Windows, выберете 64-разрядную версию
интерпретатора в диалоге установки.

Скачайте архив со скриптами L2Bot и распакуйте его:
https://github.com/ellysh/l2bot/archive/master.zip

НАСТРОЙКА
---------

Прежде всего поставьте галочку "чат с Enter" в настройках вашего клиента
Lineage.

Вы можете модифицировать перечисленные ниже конфигурационные файлы, чтобы лучше
адаптировать L2Bot для вашего клиента Lineage.

Список конфигурационных файлов из каталога 'conf':
interface.au3 - расположение окон и областей экрана для анализа цветов
control.au3 - содержит настройку панели горячих клавишь и специальные клавищи управления персонажем
targets_ru.au3,targets_en.au3 - список шаблонов для поиска мобов по имени на русском и английском языках

Следующая информация может быть указана в файле control.au3:
kAttackKey - клавиша действия /attack на панели горячих клавишь (по умолчанию F1)
kNextTargetKey - клавиша действия /targetnext на панели (по умолчанию F10)
kPickDropKey - клавиша действия /pickup на панели (по умолчанию F8)
kSitKey - клавиша сесть/встать на панели (по умолчанию F9)
kHealthPoition - клавиша использования зелья исцеления (по умолчанию F5)

Для уточнения следующей информации об интерфейсе воспользуйтесь программой CoolPix.exe из
каталога 'tools'.

Следующая информация может быть указана в файле interface.au3:
kTargetWindowPos[x1, y1, x2, y2] - координаты левой верхней точки (x1 и y1) и
правой нижней точки (x2 и y2) окна состояния цели
kTargetWindowColorBrown - первый цвет для определения наличия окна цели
(по умолчанию коричневый)
kTargetWindowColorGray - второй цвет для определения наличия окна цели (по
умолчанию серый)
kTargetHealthColor - цвет полной полосы HP цели
kTargetManaColor - цвет полной полосы MP цели (для петов/самонов)
kTargetManaEmptyColor - цвет пустой полосы MP цели (для петов/самонов)
kTargetHealthPos - координаты части полосы HP цели для обнаружения события "цель
повреждена"
kSelfHealthCritPos - координаты части полосы HP игрока для обнаружения события
"критическое значение HP"
kSelfHealthHalfPos - координаты части полосы HP игрока для обнаружения события
"осталось половина HP"
kSelfHealthEmptyPos - координаты полной полосы HP игрока
kSelfHealthColor - цвет полной полосы HP игрока
kSelfManaMinPos - координаты части полосы MP игрока для обнаружения события
"критическое значение MP"
kSelfManaColor - цвет полной полосы MP игрока
kMoveControlPos1,kMoveControlPos2,kMoveControlPos3 - координаты 3-х произвольных точек на экране для
обнаружения события "игрок двигается"

Также вы можете указать горячие клавиши использования умений специфичных для каждого класса.
Эти настройки вы можете найти в следующих файлах каталога 'tactics' - spoiler.au3, tank.au3 и warcrayer.au3.

ЗАПУСК
------

Для запуска L2Bot выполните следующие действия:

1. Запустите ваш клиент Lineage в оконном режиме (нажмите Alt+Enter для перехода
в этот режим из полноэкранного) 
2. Растяните окно клиента Lineage на весь экран
3. Зайдите в игру своим персонажем.
4. Запустите один из скриптов в корневом каталоге проекта L2Bot
5. Переключите фокус ввода на окно клиента Lineage
6. Нажмите Alt+F2 чтобы бот захватил окно клиента Lineage
7. Нажмите Alt+F3 для запуска бота
8. Для завершения работы бота нажать Alt+F1

БОТ РЫБАЛКИ
-----------

Алгоритм бота рыбалки реализован в скрипте fishing.au3 из корневого каталога проекта L2Bot.

Для настройки бота рыбалки откройте в текстовом редакторе файл fishing.au3 и укажите значения следующих переменных:
kFishingWindowPos - координаты окна "Рыбалка"
kFishingColor1, kFishingColor2 - цвета двух произвольных пикселей в окне "Рыбалка" для определения факта
наличия этого окна
kSkillWindowPos - координаты иконки умения, отображаемой как подсказка в окне "Рыбалка"
kSkillReelColor - цвет пикселя на иконке умения "Подсечь" (Reeling)
kSkillPumpColor - цвет пикселя на иконке умения "Тянуть" (Pumping)
kFishingKey - горячая клавиша для умения "Рыбалка" (по-умолчанию F1)
kSkillPumpKey - горячая клавиша для умения "Тянуть" (по-умолчанию F2)
kSkillReelKey - горячая клавиша для умения "Подсечь" (по-умолчанию F3)

Перед запуском бота рыбалки выполните следующие действия:
1. Разместите своего персонажа около водоема, где возможно применить умение "Рыбалка"
2. Наденьте удочку и любую наживку для новичков (с профессиональной наживкой бот не работает)

Запуск и остановка бота рыбалки осуществляется аналогично остальным типам ботов:
Alt+F2 - захват окна клиента Lineage
Alt+F3 - запуск бота
Alt+F1 - окончание работы бота

БОТ ЧАТА
--------

Алгоритм бота чата реализован в скрипте chat.au3 из корневого каталога проекта L2Bot.

Для настройки бота чата откройте в текстовом редакторе файл chat.au3 и укажите значения следующих переменных:
Текст сообщений и задержка между выводом сообщений для чат-бота необходимо указать в файле chat.au3:
kMessageTextRus - сообщение на русском языке (пишутся в английской раскладке)
kMessageTextEn - сообщение на английском языке
kDelayMinutes - задержка в минутах между выводом сообщений

В случае использования Java сервера Lineage необходимо сообщения на русском языке вводит в английской раскладке.
Например, слово 'Привет' будет выглядеть как 'Ghbdtn'. Кроме того английский и русский должны быть указаны
в разных переменных - kMessageTextEn и kMessageTextRus соответственно.

Если используется официальный сервер Lineage или C++ сборка сервера (например rpg-club) 
никаких дополнительных условий для ввода русского текста сообщений нет. В этом случае для английского и русского
текста сообщений следует использовать только переменную kMessageTextEn, а kMessageTextRus оставить пустой.
